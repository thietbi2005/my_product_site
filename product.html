Bending Cold Punch

Application:

HSTUTM-BTC bending cold punch set is a special tool necessary for cold bending Test of steel bar. It can be used together with Universal Testing Machine or bending machine.

Technical Parameters

  • Number of punch: 32
  • Number of punch block: 4
  • Diameter of punch: 6~128mm
  • Scope of application: 6~32mm diameter
  • Weight: around 45kg
  • Packaging dimension: 650×370×135mm
