HST-N50ZW Torsion Analysis Testing Machine

This machine is suitable for Notebook's Hinge automatic swing life Test. Through computer analysis, it can accurately measure the relative response curve of Hinge's torque and Angle, and accurately control the swing Angle, speed, setting times and pause time. The biggest feature is that you can do the swing test, grab the torque and Angle change, without the need to be measured in the disassembly, to achieve the actual simulation test, in addition, this machine uses servo motor and Windows view control system, easy to operate, and all the data can be stored, is definitely a reliable high-performance Testing instrument.
