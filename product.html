HRS-150D-Z Automatic Rockwell Hardness Tester

This model Hardness Tester is an automatic Rockwell Hardness Tester. After place on the specimen, the whole Test process will be completed automatically, only press one key, the hardness value will be readout. Main Features: 1. Specimen will be raised up automatically (without height limited) 2. Automatically load the initial test force 3. Automatically load the main test force 4. Automatically measure the hardness values 5. Hardness value automatically be digitally displayed 6. May test with such Rockwell scales as HRA, HRB, HRC, HRD, HRE, HRF, HRG, HRH, HRK 7. Automatically revise the measuring errors 8. Automatically hardness exchange 9. Print the test result with a built-in printer Main Technical Specifications：
Exchange Scales Superficial Rockwell, Brinell, Vickers
Readout Hardness Value Large Digital Display Screen
Max Height of Specimen 175mm, Added Height: 400mm
Data  Output Built-in Printer, RS232 Interface
Dwelling Time 0~60s
Carry Out Standard Chinese Standard GB/T230.1, GB/T230.2, JJG112 Inspection Rule, ASTM Standard
Distance between Indenter to outer wall 160mm
Specimen Auto- rising Speed 3mm/s
Initial Test Force 10kgf(98.07N)
Test Force 60kgf(588N), 100kgf(980N),  150kgf(1471N)

Hardness Test Range HRA:20-88, HRB:20-100, HRC:20-70, HRD:40-77, HRF: 60-100, HRG：30-94, HRH：80-100, HRK：40-100
