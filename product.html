P-2T Metallographic specimen polishing machine

Application

: Model P-2T metallographic specimen polishing machine is a kind of double-disc table type polishing machine and can be operated by two persons at the same time. It is suitable for the polishing of the specimen which has been grinded. The specimen surface is very smooth after processing and can be used to observe and measure the Metallographic structure of specimen under microscope.

Main specification

Model P-2T
Diameter of polishing disc 200mm,
Polishing disc rotating speed 1400 rpm
Cooling method water cooling
Motor 180W
Power supply Single phase, 220V, 50Hz
Net weight 35 Kg
Dimension 700×410×320 mm
