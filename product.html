Q-100B metallographic sample cutting machine

Application

: Model Q-100B  manual / automatic Metallographic specimen cutting machine is equipped with big cutting room and movable T-shape work table. Therefore this machine has the ability of cutting rectangle and large specimens. Also the equipped separate type vice is very convenient for operator to clamp various heterotypic work pieces. Various cutting data can be displayed on high definition backlight type LCD screen, manual cutting and automatic cutting means can be switched at will. Large cutting chamber, tempered glass observing window and it is equipped with 60L cooling liquid water tank. Non-burning loss swing type cutting means that improved the cutting quality. Automatic withdraw function after cutting is finished, bidirectional feeding means increase the length of deep cutting.
