Rail Static Bend Testing Machine

GGW Series Servo-hydraulic Rail Static Bend testing machine designed for Testing of rails after welding; analysis is performed at the edge of the weld collar of an aluminothermic weld, in order to examine one of the failure modes that occur in this weld type under high axle load conditions Using the GGW Rail Static Bend Testing machine during the installation of rail allows the user to periodically Test the quality of welds.
