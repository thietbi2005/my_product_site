MOPAO 1T Metallographic Specimen Grinding And Polishing Machine

Application

: MOPAO 1T  is a touch screen type Metallographic specimen grinding and polishing machine which is equipped with single disc and features stepless speed changing. It can realize the whole process of rough grinding, fine grinding, rough polishing and finishing polishing for specimen preparing. This machine features easy operation and economical applying. It is the ideal specimen preparing instrument for using in factories, scientific research institutes and laboratories of colleges. Technical specifications
Model MOPAO 1T
Rotating speed of working disk 99-1000 rpm (stepless speed changing)or 150 / 300 / 600 / 800 rpm(four-level constant speed)
Diameter of working disk 203 mm (250mm is optional)
Power supply 220V 50Hz
Dimensions 800 x 440 x 360 mm
Net weight 41kg
