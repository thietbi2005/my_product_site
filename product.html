ONE1-M5 Video Extensometer For Tension Test

Application:

Video Extensometer is an optical non-contact type three-dimentional deformation measuring system based on DIC(Digital Image Correlation)measurement method.

Features:

l Default single cam measurement length:110mm (220mm)

l Default resolution according to lSO9513: Class 0.5 (Class 1)

l Default resolution according to ASTME83-10: Class B-1(Class B-2)

l Gauge Length: Selectable single or multiple points L0

l Data acquisition rate: 75-500Hz

l Transverse Measurement: Optional transverse or Bi-axial

l Axial and radial neck detection

l Output:Digital and analog

l Lighting:Auto switching monochromatic light

l Software Integrated: via extensometer user interface**

l Mounting: Test frame,chamber or Tri-pod
