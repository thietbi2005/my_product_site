HST-TNP2 Computer Controlled Spring Torsion Fatigue Testing Machine

This microcomputer-controlled spring torsional fatigue Testing machine is mainly used for torsional life testing of various torsion springs, scroll springs and spring assemblies. It can also be used for torsional performance Testing of other elastic components and friction mechanisms. The Tested specimen is installed between the fixtures with adjustable space, which can be adapted to the test of different torsional fatigue mechanical properties of the tested specimen. The test bench adopts computer control operation and can collect data such as torque, peak value, angle, amplitudenumberfrequencyfatigue curve; the display shows the life fatigue test curvetorque-time curve of the tested part; it can realize test data storage, Obtain and print test reports, etc.

This spring torsional fatigue testing machine is mainly used in spring manufacturers, electrical appliance manufacturers, power machinery factories, colleges and universities and related scientific research institutions. It can automatically complete the entire cycle test environment of the spring, and can also independently measure the torsional amplitude and torsional frequency of the spring. The user can adjust the test program according to actual needs during use.
