HST-RMD300 Rubber No Rotor Rheometer

1.Applicable Industry:

It is suitable for Testing the variation of shear modulus when raw rubber is vulcanized. The shear modulus is proportional to the crosslinking density. So the results reflect the changes in the degree of crosslinking during vulcanization. So that we can measure the parameters like curing speed, Curing time and desulfurization. 2.Meet standard: GB/T16584— use rotorless curemeter to Test vulcanization characteristics ISO6502 Vulcanizing machine standard data ASTMD5289 Rubber properties test methods
