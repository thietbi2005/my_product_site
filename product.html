WDL Horizontal Electronic Tensile Testing Machine

Application:

Electronic Tensile Testing Machine consists of the host, drive system and computer measurement and control system of three parts. This machine is the automatic computer acquisition experiment data and the automatic control process. Frame use the H frame structure, Test force by the ball screw, load sensor, the encoder measuring displacement, computer screen displays real-time test power and test curve and automatic processing of test data according to the preset test method, sample clamping way is bolt method.
